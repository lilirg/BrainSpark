# 项目启动前置准备清单

---

## 一、 学术与专业内容准备（核心壁垒，最优先）

认知测评产品**最忌讳“伪科学”**，在开发前必须夯实专业基础，这是AI生成高质量建议的“燃料”。

* [ ] **1. 确定 MVP 测评量表与常模数据来源**
  * **准备内容**：确定首发的 2-3 个测评（如舒尔特方格、数字广度）。
  * **关键动作**：获取这些测评的**中国儿童常模数据**（不同年龄段的平均值、标准差）。如果没有常模，测出来的分数就毫无意义。建议联系高校心理学实验室合作，或查阅权威学术文献提取基础常模。
* [ ] **2. 构建《认知-学业映射与干预策略知识库》**
  * **准备内容**：一份结构化的 Excel/文档。这是 AI 大模型 RAG（检索增强）的核心知识库。
  * **字段示例**：`认知维度` (如：工作记忆) -> `学业表现影响` (如：数学多步计算易出错、英语长句阅读困难) -> `家庭干预策略` (如：复述购物清单、玩记忆翻牌) -> `在校学习建议` (如：将长任务拆分为短任务)。
* [ ] **3. 落实专家顾问背书**
  * **准备内容**：寻找 1-2 位发展心理学或教育学背景的专家（高校教师或资深教研员）作为顾问。
  * **关键动作**：由专家审核您的测评逻辑和干预策略库，并在产品上线时提供“专家联合研发”的背书，极大提升家长信任度。

## 二、 产品与设计准备（研发的依据）

不要直接让开发“边想边写”，必须有清晰的蓝图。

* [ ] **1. 输出高保真产品原型 (Figma / 墨刀 / Axure)**
  * **学生端 (Web)**：重点设计游戏化测评的交互流程、动效反馈、防打扰机制。
  * **家长端 (小程序/H5)**：重点设计**AI 评估报告的可视化呈现**（雷达图、解读文案、行动建议），这是家长付费的核心触点。
* [ ] **2. 撰写核心 PRD (产品需求文档)**
  * **准备内容**：详细说明业务流转规则。特别是**计费逻辑**（免费测几次？报告怎么解锁？）和**防沉迷逻辑**（超时如何强制中断）。
* [ ] **3. 制定《前端高精度行为埋点字典》**
  * **准备内容**：与前端技术负责人共同定义。明确舒尔特方格需要采集哪些数据（如：`click_x`, `click_y`, `reaction_time_ms`, `hover_time`），统一 JSON 数据格式。

## 三、 合规与法务准备（生命线，不可省略）

K12 + 未成年人 + AI，这三个标签叠加，合规要求极高。

* [ ] **1. 起草核心法律协议**
  * **准备内容**：找专业互联网律师起草《用户服务协议》、《隐私政策》，**特别是《儿童个人信息保护规则》**（必须单独成文，明确收集目的、存储方式、家长权利）。
* [ ] **2. 建立《对外宣传红线词库》**
  * **准备内容**：整理一份禁用词文档，分发给所有产品和运营人员。
  * **绝对禁用**：天赋、智商、测智商、天才、治疗、多动症、抑郁症、100%提升等。
  * **替换话术**：认知潜能、学习力评估、专注力发展、科学干预等。
* [ ] **3. 资质与备案规划**
  * **准备内容**：注册公司主体，准备申请 ICP 备案（网站必须）、软著（上架应用商店或保护知识产权必须）。评估是否需要做“生成式 AI 算法备案”（早期若仅调用大厂已备案 API 且不涉及社会舆论，可暂缓，但需咨询当地网信办）。

## 四、 技术与基础设施准备（开发的基石）

为技术团队扫清障碍，提供“弹药”。

* [ ] **1. 采购与开通基础云服务**
  * **准备内容**：购买域名并完成备案；购买云服务器（阿里云/腾讯云）；购买对象存储（OSS/COS，用于存头像和报告 PDF）。
* [ ] **2. 申请大模型 API 与第三方服务**
  * **准备内容**：注册并认证百度文心一言 / 阿里通义千问 / 智谱 GLM 等大模型的企业开发者账号，获取 API Key，并充值测试费用。
  * **其他服务**：开通短信验证码服务、微信支付/支付宝支付商户号。
* [ ] **3. 搭建研发协同环境**
  * **准备内容**：建立 Git 代码仓库（GitLab/Gitee）；搭建项目管理工具（TAPD / 飞书项目 / Jira）；建立 UI 设计交付平台（蓝湖 / MasterGo）。

## 五、 团队与预算准备（找对人，算好账）

* [ ] **1. 明确 MVP 阶段团队编制与 JD**
  * **准备内容**：根据前面的架构，写出招聘需求。
  * **核心必招**：1名懂游戏化交互的前端（熟悉 Vue3 + PixiJS/Canvas）、1名全栈或后端主力（Java/Go）、1名产品经理。AI 算法早期可由后端兼岗或直接使用 Prompt 工程师。
* [ ] **2. 制定前 6 个月的资金预算表**
  * **准备内容**：一份详细的 Excel 预算表。
  * **包含科目**：人力成本（工资+社保）、云资源及 API 调用成本（大模型 Token 费用早期需预估）、专家顾问费、法务咨询费、早期种子用户获取成本（如赠送小礼品、投放测试）。
